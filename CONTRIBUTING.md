# Contributing

We welcome researchers and developers to help validate and extend our hashtag-based social verification system.

## Quick Start

1. **Clone and setup**: `git clone [repo] && cd hashtag-blockchain-proof-of-concept`
2. **Install dependencies**: `pip install -r requirements.txt` (when available)
3. **Run sample demo**: `./examples/sample_run.sh` (coming soon)

## Areas We Need Help

- **Reproducibility**: Validate parsing & geographic clustering scripts on sample data
- **Statistical validation**: Independent assessment of accuracy metrics and error analysis
- **Security audit**: Sybil/collusion resistance and platform manipulation testing
- **Ethics & compliance**: Privacy review and platform ToS compliance analysis
- **Scalability testing**: Performance analysis and optimization

## How to Contribute

1. **Open an issue** with label `research-collab` describing your area of interest
2. **Email shannon@loyal9.app** for access to sample datasets or collaboration discussion
3. **Check existing issues** for `good-first-issue` or `help-wanted` labels

## Issue Labels

- `research-collab`: Academic collaboration requests
- `reproducibility`: Validation of methods and results  
- `security-audit`: Security analysis and testing
- `data-request`: Access to sample datasets
- `good-first-issue`: Beginner-friendly contributions

## Code of Conduct

Be respectful, constructive, and focused on advancing the research. This is an academic collaboration.
